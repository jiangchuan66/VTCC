# VTCC AI Development Guide

## Project

VTCC 是一个微信小程序。

目标：

帮助 Valorant 玩家快速查询 VCT 官方赛事信息。

这是一个个人作品集项目。

重点展示：

- 产品设计能力
- 工程能力
- 前后端开发能力
- AI 协作开发能力

---

## Coding Rules

所有代码必须：

- 模块化
- 可维护
- 可读
- 不重复

禁止：

- 超长函数
- 重复代码
- 魔法数字

# VTCC Development Roadmap

## 项目目标

VTCC 是一个面向 Valorant（无畏契约）电竞观众的 VCT 赛事信息查询微信小程序。

本项目主要作为个人求职作品集，重点展示：

* 产品需求分析能力
* UI 与交互设计能力
* 微信小程序开发能力
* Node.js 后端开发能力
* 数据库与 API 设计能力
* Git 与 GitHub 协作能力
* AI 辅助开发与项目管理能力

---

## 开发原则

* MVP 优先，先完成核心查询功能
* 每次只开发一个可独立验收的任务
* 每个任务完成后进行测试、Commit 和 Push
* 前期使用 Mock 数据，后期再接入真实数据
* 不在 MVP 阶段开发社区、评论、资讯、竞猜等功能
* 所有重要决策和变更必须记录在文档中

# Git 工作流

每完成一个独立任务：

```bash
git status
git add .
git commit -m "本次修改说明"
git push
```

常用 Commit 类型：

```text
chore: 项目配置或工具调整
docs: 文档修改
feat: 新增功能
fix: 修复问题
refactor: 重构代码
test: 增加或修改测试
style: 页面样式调整
```

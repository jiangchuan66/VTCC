# VTCC

> 一个面向 Valorant（无畏契约）VCT 赛事的数据查询微信小程序。

## 项目简介

VTCC 是一个帮助玩家快速查询 VCT 赛事、比赛、战队和选手信息的小程序。

本项目作为个人作品集项目开发，采用 AI 协作开发流程，完整展示从产品设计、UI 设计、数据库设计到前后端开发的全过程。

## 技术栈

### 前端
- 微信原生小程序
- JavaScript
- WXML
- WXSS

### 后端
- Node.js
- Express

### 数据库
- MySQL
- Redis

### AI 工具
- ChatGPT（产品规划）
- Claude Code（架构设计）
- Codex（开发）
- Open Design（UI 设计）

## 项目结构

```text
docs/        产品文档
frontend/    微信小程序
backend/     Node.js 后端
prompts/     AI Prompt
```
